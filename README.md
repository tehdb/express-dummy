express-seed
==============

get started with express/nodejs/coffee-script/grunt

prerequisites
--------------

- Node.js http://nodejs.org/
- Grunt http://gruntjs.com/
- CoffeeScript http://gruntjs.com/
- nodemon https://github.com/remy/nodemon

local packages
--------------
```bash
npm install
```

start server
--------------
```bash
npm start
```
